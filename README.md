# Portfolio-Optimization
Portfolio optimization is the process of selecting the best portfolio, out of the set of portfolios being considered, according to some objective. The objective typically maximizes factors such as expected return and minimizes costs like financial risk.

Stock analysis is a method for investors use (short or long term) and traders to make buying and selling decisions. By studying and evaluating past and current historical data, fundamental data or news, investors and traders attempt to gain an winning edge in the markets by making the best decisions.
